# Laureley - Modern Perfume Brand Landing Page

Laureley adalah landing page modern dan elegan untuk brand parfum fiksi, dirancang dengan tema **"Deep Ocean"** (Dark Navy & Neon Sky Blue). Website ini dibangun menggunakan **Next.js (App Router)** dan **React Bootstrap**, menawarkan pengalaman pengguna yang responsif dengan navigasi satu halaman (one-page scroll).

* **Neon Effects:** Custom CSS untuk efek *glowing* pada teks, border, dan tombol.

## üöÄ Cara Menjalankan Project

Ikuti langkah-langkah ini untuk menjalankan proyek di komputer lokal Anda:

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/kaminglo/laurel.git]
    cd laurel
    ```

2.  **Install Dependencies**
    Pastikan Anda sudah menginstall Node.js.
    ```bash
    npm install
    # atau
    yarn install
    ```

3.  **Install Bootstrap Icons (Jika belum)**
    ```bash
    npm install bootstrap-icons
    ```

4.  **Jalankan Server Development**
    ```bash
    npm run dev
    ```

5.  **Buka Browser**
    Buka [http://localhost:3000](http://localhost:3000) untuk melihat hasilnya.
